# 🐱 Kedi Tamagotchi

Browser uzerinde calisan pixel art kedi bakim oyunu!
Phaser.js 3 ile yapilmis, tamamen programatik pixel art — harici dosya gerekmez.

## 🎮 Oyna
> Linki GitHub Pages deploy sonrasi guncelleyin.

## ✨ Ozellikler

### Kedi Bakimi
- 🎨 8 farkli kedi rengi (turuncu, gri, siyah, beyaz, kahverengi, tekir, pembe, mavi)
- 🐱 Kedi buyume sistemi (yavru → genc → yetiskin)
- 👗 9 aksesuar (sapka, tac, fiyonk, cicek tac, papyon, tasma, atki, gunes gozlugu, gozluk)
- 📊 5 stat bari (aclik, mutluluk, enerji, temizlik, eglence)
- 🚽 Kedi tuvaleti sistemi
- 🌙 Gece/gunduz modu (uyku sirasinda pencereden ay ve yildizlar)
- 💾 Otomatik kayit (LocalStorage)
- 🐾 Kedini tikladigin yere yurut
- 📋 Kedi kimlik karti (isim, dogum tarihi, yas, istatistikler)
- 😴 Kedi yataginda uyur, ruya gorur

### Mini Oyunlar (9 + 1 adet)
- 🐟 Balik Yakalama
- 🐭 Fare Yakalama
- 🃏 Hafiza Karti Eslestirme
- 🧵 Yumak Sarma
- 🧱 Kedi Tetris
- 🪰 Sinek Yakalama
- 🥛 Sut Doldurma
- 🎁 Kutu Ac (Mystery Box)
- 🐦 Kus Gozleme
- 💭 Ruya Mini Oyunu (uyku sirasinda)

### Ses & Muzik
- 🎵 8-bit lofi arkaplan muzigi
- 🎮 Upbeat mini oyun muzigi
- 🔊 10+ ses efekti (tumu Web Audio API ile programatik)
- 🎛️ Ayri muzik ve efekt ses kontrolleri

### Oda Dekorasyonu
- 🪟 Pencere (gunes, ay, yildizlar)
- 🕐 Duvar saati
- 📚 Duvar rafi (saksi cicek, kitaplar, kedi heykeli)
- 🖼️ Balik tablosu
- 🧶 Oyuncak sepeti
- 🐈 Kedi agaci (3 katli)
- 🛏️ Kedi yatagi
- 🍖 Mama ve su kasesi
- 🧹 Kedi tuvaleti

## 🛠️ Teknoloji
- **Phaser.js 3** — 2D oyun motoru
- **Vanilla HTML/CSS/JS** — framework yok
- **Web Audio API** — programatik 8-bit sesler
- **Programatik Pixel Art** — tum gorseller kod ile uretilir, harici asset dosyasi yok
- **LocalStorage** — oyun kayit sistemi

## 🎯 Nasil Oynanir
1. Oyunu ac, kedine isim ver ve renk sec
2. Kedini besle, sev, banyo yap, oyun oyna!
3. Statlari yuksel tut — kedi mutlu olsun
4. Mini oyunlar oynayarak eglence ve mutluluk kazan
5. Kedini uyut, ruya oyunu oyna
6. Aksesuarlarla kedini susle
7. Kedi buyudukce degisir — yavru, genc, yetiskin!
